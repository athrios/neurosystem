# Adaptadores de testes para o pré-laudo

Os resultados calculados de SCARED, SDQ, SNAP-IV e QEDP passam por adaptadores
antes de entrar no pré-laudo.

Cada adaptador fornece:

- nome completo para o procedimento;
- síntese editável para a análise dos resultados;
- relação dos escores e classificações para resultados dos formulários;
- dados numéricos para o gráfico de perfil.

Os textos não convertem resultados de rastreio em diagnóstico e incluem a
orientação de integração com entrevista, observação clínica e demais
instrumentos.

Pré-laudos novos recebem o conteúdo automaticamente. Em documentos já salvos,
o profissional deve usar **Atualizar conteúdos automáticos**; isso preserva
edições manuais até que a atualização seja solicitada.
